# Sito Personale

Homework del corso di Tecnologie per il Sito Web (TSW) per la **creazione** di un sito personale, in continuo aggiornamento.

## Descrizione

Questo progetto consiste in un semplice sito statico sviluppato in HTML e CSS, pensato come biglietto da visita personale.  
Include una homepage, una pagina di presentazione personale e una pagina contatti.

Le principali pagine sono:

- `index.html`: pagina principale del sito.
- `whoami.html`: pagina “Chi sono” con informazioni personali e/o curriculum.
- `contacts.html`: pagina per i contatti (email, social, form, ecc.).

Le risorse grafiche e di stile sono organizzate nelle cartelle:

- `css/`: fogli di stile CSS.
- `images/`: immagini utilizzate nel sito.

## Tecnologie utilizzate

- HTML per la struttura delle pagine.
- CSS per la formattazione e il layout.

## Obiettivi didattici

Il progetto nasce come esercitazione universitaria con i seguenti obiettivi:

- Progettare la struttura di un sito personale.
- Organizzare file e cartelle in modo chiaro (HTML, CSS, immagini).
- Applicare regole di stile CSS per migliorare usabilità e presentazione.

## Come visualizzare il sito

1. Clona la repository:
   ```bash
   git clone https://github.com/C-Vollono/SitoPersonale.git
   ```
2. Apri `index.html` con il tuo browser preferito.

## Stato del progetto

Il sito è in fase di sviluppo ed è soggetto a modifiche e miglioramenti nel tempo (“in continuo aggiornamento”).
